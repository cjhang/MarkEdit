# MarkEdit

MarkEdit is a free and **open-source** Markdown editor for macOS, built by [Ying Zhong](https://github.com/cyanzhong). See also the original project [here](https://github.com/MarkEdit-app/MarkEdit)

# Changes

The changes reflect my personal preference and consist of two main updates:

1. **Quick Tab Switching** – `Command + number` now selects and switches tabs rather than creating headings. Headings can still be created easily by typing `#`.

2. **Image Hover Preview** – Markdown is optimized for text, not for displaying images. This limitation is addressed with an image-hover feature: hovering the mouse over an image URL opens a pop‑up preview of the image.

If you're interested in applying these changes, please refer to `CHANGES.md`.
